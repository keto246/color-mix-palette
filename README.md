# 色まぜパレット 🎨

赤・黄・青の3原色タイルを盤面に置き、隣同士で混色させて**目標の配色パターンを再現する**ロジックパズルです。全36ステージ。ブラウザ（PC・スマホ）で遊べます。

## 遊び方

- 手持ちの色タイルを選んで、空きマスをタップして置く
- タイルを**となり合わせると自動で混ざる**
  - 赤 + 黄 = 橙 ／ 黄 + 青 = 緑 ／ 赤 + 青 = 紫
  - **3色そろうと茶（失敗色）** になるので注意
- 一次色（赤・黄・青）のタイルは**長押しで回収**できる
- 置いたタイルは隣の空マスへ**スライド**できる（タイルを選んで移動先の空マスをタップ）
- 決められた手数以内にクリアすると**星（最大3つ）**がもらえる

### ギミック

| 記号 | 名前 | 効果 |
|---|---|---|
| 🔒 | 固定タイル | 動かす・回収はできないが混色には参加する |
| ⬜ | 白マス | 置けず、混色にも参加しない壁 |
| 🔀 | ワープマス | つながったマス同士が隣接扱いになる |
| 💧 | 滲みマス | 置くと隣の空マス1つへ色が広がる |
| 🪞 | 鏡 | 置くと左右対称の位置にも複製される |
| 🧪 | 触媒マス | そのマスが絡む混色の結果を色相環で1段ずらす |

## 混色ルールの実装メモ

仕様検討時、「隣接する同じ色で混色」と「A+B→C の連鎖」という2つの表現が混在していたため、
パズルとして破綻なく成立する以下のルールに**確定**しています（詳細は [`src/game.ts`](src/game.ts) 冒頭コメント）。

- 各タイルは「含む原色の集合」を持つ（例：橙 = {赤, 黄}）。
- タイルを置く／スライドした瞬間に、対象と隣接タイルの原色を**和集合**した色へ**両方とも**変化する。
- 変化したタイルから連鎖的に混色判定が伝播する。
- 原色は和集合なので単調増加し、必ず安定状態（最大は茶）で停止する。

ステージは「空盤面に対して実際の `place()` を前進実行して構築」しているため、
生成された目標は**構成的に必ず解ける**ことが保証されます（[`src/stages.ts`](src/stages.ts)）。

## 開発

```bash
npm install      # 依存インストール
npm run dev      # 開発サーバー（http://localhost:5173/color-mix-palette/）
npm test         # ユニットテスト（vitest）
npm run lint     # ESLint
npm run build    # 本番ビルド（dist/）
```

### 技術スタック

- TypeScript / Vite
- 効果音は Web Audio API でその場合成（音源ファイル不要）
- 進捗・スター評価は LocalStorage に保存
- 依存ライブラリなしのバニラ実装

## デプロイ（GitHub Pages）

`main` ブランチへの push で GitHub Actions が自動ビルド＆デプロイします（[`.github/workflows/deploy.yml`](.github/workflows/deploy.yml)）。

1. GitHub に `color-mix-palette` という名前でリポジトリを作成
2. `git remote add origin https://github.com/<ユーザー名>/color-mix-palette.git && git push -u origin main`
3. リポジトリの **Settings → Pages → Source** を「GitHub Actions」に設定

> リポジトリ名を変える場合は [`vite.config.ts`](vite.config.ts) の `base` も合わせて変更してください。
> 公開後、[`index.html`](index.html) の `og:image` / `og:url` を実際の公開URLに置き換えると SNS シェアのサムネイルが表示されます。
